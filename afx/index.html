What is AFX?

Designed from the ground-up to be fully customizable, AFX offers a complete, joystick controlled system for accessing audio, video, emulators, pictures, and other media.

Because it's written in Python, AFX can be extended to support virtually any kind of application. We kept non-programmers in mind to keep the plugin system easy, and programmers can unlock the true potential of the system.

Why?

AFX's primary purpose is for a tv-connected, kiosk-like system. Build a mini-itx system with tv-out, attach some USB controllers, install AFX and use it like you would any standard videogame console.

Who?

The AFX Project was started by Sadistech and was written by Spike Grobstein. New developers are always welcome, whether for direct development of AFX or for creating additional plugins, artwork, how-tos, etc.
